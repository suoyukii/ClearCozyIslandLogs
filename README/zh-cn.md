中文名：清除舒舒服服小岛时光的日志

游戏 舒舒服服小岛时光 在游玩后会产生两个日志文件，这两个日志的文件体积会一直在增大。

打开 ClearCozyIslandLogs，就会自动删除 舒舒服服小岛时光 产生的日志。

## ✨ 特点

- 自动化
- 轻量级软件
- 释放硬盘空间
